# Trae

## Introduction

**Trae** (字节跳动) is an AI-powered code editor developed by **ByteDance** — the company behind TikTok. It is positioned as a free, China-friendly alternative to Cursor, offering a familiar VS Code-like interface with integrated AI assistance. Trae is available in two editions:

- **Trae CN** (`trae.com.cn`) — Optimized for mainland China, powered by ByteDance's **Doubao** / **GLM** models
- **Trae International** (`trae.ai`) — Global edition, powered by **Claude** and **GPT-4o**

> Trae CN 是字节跳动开发的一个类似 Cursor 的 AI 代码编辑器，可以作为 Cursor 的国内平替。

优点：
- 提供 **免费** 较稳定的 **GLM-5** 模型，还有其他主流模型也免费提供
- 免费用户能使用绝大部分需要的功能
- 稳定不封号

缺点：
- 有点笨，需要耐心和锤炼自己的勇气（更新：搭配 GLM-5 已经不笨了）

## Key Features

### AI Chat & Inline Editing
A side-panel AI chat and `Cmd+K` / `Ctrl+K` inline editing — ask questions, generate code, or explain selected code, all within the editor.

### Builder Mode (Agent)
An autonomous agent mode where Trae can read files, make multi-file edits, run terminal commands, and iterate toward a goal — similar to Cursor Composer.

### Free Premium Models
Trae CN offers free access to:
- **GLM-5** — ByteDance's capable model, significantly improved over earlier versions
- **DeepSeek** — available on the free tier

Trae International offers:
- **Claude 3.7 Sonnet** — free tier access
- **GPT-4o** — free tier access

### VS Code Compatibility
Built on VS Code's core, supporting most VS Code extensions, themes, and keybindings — easy migration for VS Code users.

### MCP Support
Trae supports the **Model Context Protocol (MCP)**, allowing integration with external tools and data sources.

## Tips & Best Practices

- **Use Builder Mode for big tasks** — For multi-file changes, switch to Builder (Agent) mode for better results.
- **GLM-5 for everyday coding** — GLM-5 is now capable enough for most coding tasks; save Claude/GPT credits for complex reasoning.
- **Use `.traerules`** (similar to `.cursorrules`) to give Trae persistent instructions about your project style.
- **International edition for English projects** — The international edition with Claude/GPT tends to handle English-language codebases and documentation better.
